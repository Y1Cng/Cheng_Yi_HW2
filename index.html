OOP Counter with State-Driven UI

MMED-3014 HW2 | By Cheng Yi

Overview

The Counter class is a reusable OOP web component for managing numeric counts with state-driven UI interactions. It enables modular, independent counter instances on a single page with core count manipulation features.

The StepCounter subclass extends the base class to demonstrate OOP inheritance and polymorphism, adding a configurable step value for custom increment/decrement behavior.

How To Use

1. HTML Container Markup

Add empty container elements for counter instances with unique IDs:

<div id="counter-default" class="counter"></div>
<div id="counter-custom" class="counter"></div>
<div id="counter-step" class="counter"></div>

2. JavaScript Instantiation

Import the classes and create instances with ES6 modules (no DOMContentLoaded required):

import { Counter } from './js/counter.js';
import { StepCounter } from './js/StepCounter.js';

new Counter('#counter-default');
new Counter('#counter-custom', 5);
new StepCounter('#counter-step', 10, 5);

3. Parameter Explanation

  • selector (string): Required CSS selector for the target counter container
  • initialValue (number): Optional starting count (default: 0, non-negative)
  • step (number): Optional increment/decrement step (StepCounter only, default: 1)
